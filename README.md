# Sistem Manajemen Tekstil

## Deskripsi
Sistem manajemen tekstil berbasis web dengan fitur e-commerce dan dashboard statistik untuk industri tekstil.

## Fitur Utama
- Dual Role System (Supplier & Pembeli)
- Manajemen Produk (Tambah, Edit, Hapus, Upload Gambar)
- Pembelian Produk dengan validasi stok
- Tracking Pengiriman (5 status)
- Dashboard Statistik dengan Grafik Chart.js
- Statistik Penjualan per Provinsi
- Filter Pencarian Produk Lengkap

## Teknologi
- PHP
- MySQL
- Bootstrap 4
- Chart.js

## Cara Install
1. Clone repository ini
2. Import database `db_tekstil.sql` ke phpMyAdmin
3. Jalankan di localhost
